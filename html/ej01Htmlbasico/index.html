    1. Dar un par de ejemplos de uso de etiquetas meta.
      Meta de "metainformation": sirve para aportar información sobre el documento y generalmente se codifican en la cabecera de un documento html. Ejemplos:
      name="description", es una breve descripción de la página.
      name="author" Poder ingresar el nombre del autor.

    2. Nombrar al menos 2 atributos aplicables a la etiqueta HTML
      lang: "agregar el lenguaje del documento html".
      id: "Sirve para identificar de forma individual a cada elemento HTML".
      style: Define el estilo de un elemento HTML y puede determinar, por ejemplo, su color, fuente o tamaño.

    3. Como se relaciona un header http con la etiqueta head de HTML?
      Ambos contienen metadatos. Mientras que el elemento head provee metadatos para el documento, header contiene un grupo de elementos introductorios o de navegación.

    4. Que es UTF8?
      UTF8 son las siglas de "8-bit Unicode Transformation Format", un formato de codificación de caracteres Unicode e ISO 10646 que utiliza símbolos de longitud variable. Tiene una serie de características que lo hacen perfecto para codificar tu correos electrónicos y crear tu página web.

    5. Que atributo HTML sirve para aplicar estilo, a que etiqueta es aplicable?
      El elemento encargado de indicar la información de estilo es la etiqueta "Style".

    6. Donde se indica que un enlace debe ubicar el documento en una nueva pestana?
      En la etiqueta < a >, con atributo target="_blank" abre el enlace en una nueva pestaña o ventana del navegador.

    7. Explicar cuando usarian etiquetas semanticas y con que proposito.
      Las etiquetas semanticas sirven para definir el esquema principal del documento, como HEADER, ARTICLE, SECTION, ASIDE, FOOTER, NAV. Todas esas etiquetas semanticas nos indican que es el contenido que engloban y cuál es su relación con el conjunto de elementos del documento HTML

    8. ¿Que atributos de estilo basicos aplicarían a una tabla? Explicar.
      background-color: cambia el color de fondo de la celda.
      border-color cambia el color de los bordes.
      style="width": darle ancho a las celdas.

    9. ¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?
      Un campo de cuerpo de mensaje opcional ('body') lleva los datos asociados con la petición (como contenido de un formulario HTML), o los archivos o documentos asociados a una respuesta. La etiqueta "body" en html representa el contenido de un documento HTML.

    10. ¿Que atributos marcan la agrupación de celdas de una tabla?. ¿A que etiquetas se aplican?
      "Colspan" para columnas y "rowspan" para filas. Aplican a la etiqueta < td >.


    La etiqueta main es en HTML5 una etiqueta semantica. Esto quiere decir que tiene que ver con el significado de lo contenido dentro de ella. Define el contenido principal de la pagina. Unicamente puede haber una sola etiqueta como esta en todo el documento. El contenido escrito aqui dentro no se repite en el resto del sitio (todo lo contrario a los encabezados de pagina, los menues de navegacion y los pie de pagina).
